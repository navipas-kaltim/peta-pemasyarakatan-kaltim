# 📍 Peta Kerja Kaltim

Peta Kerja Kaltim adalah aplikasi **peta interaktif berbasis web** yang menampilkan lokasi Unit Pelaksana Teknis (UPT) di wilayah Kalimantan Timur.

---

## 🧑‍💻 Untuk Pengguna

Aplikasi ini dirancang untuk memudahkan pengguna dalam mencari dan mengakses informasi layanan UPT.

### ✨ Fitur Utama

* 🗺️ **Peta Interaktif**
  Menampilkan lokasi UPT secara visual.

* 📍 **Klik / Ketuk Lokasi**
  Tekan pada titik UPT untuk melihat detail informasi.

* 🏢 **Informasi Layanan**
  Setiap UPT menampilkan:

  * Nama UPT
  * Jenis layanan
  * Jadwal layanan

* 🌐 **Media Sosial & Kontak**
  Akses langsung ke media sosial atau informasi tambahan.

* 🔎 **Pencarian & Filter**
  Memudahkan menemukan UPT tertentu dengan cepat.

## 🛠️ Untuk Developer

### 📦 Teknologi

Project ini dibangun menggunakan:

* HTML, CSS, JavaScript (Vanilla)
* Library peta: Leaflet
* Data: Google Spreadsheet (CSV)

---

### 🌍 Deployment

Project ini di-host menggunakan **GitHub Pages**.

---

### 🗂️ Sumber Data

Data UPT diambil dari **Google Spreadsheet**.

Jika ingin melakukan perubahan data:

1. Edit langsung melalui Google Spreadsheet
2. Data akan otomatis ter-update pada aplikasi setelah reload

---

### ⚠️ Catatan Penting

* Jangan mengubah struktur kolom pada spreadsheet tanpa koordinasi
* Pastikan format data tetap konsisten (terutama latitude & longitude)
